카레를 만드는 방법

  1. 야채를 깍뚝썰기해서 준비하기.
  2. 카레를 물에 풀어준다.
  3. 카레를 끓이고 썰허놓은 야채를 넣는다.
  4. 완성되면 밥 위에 뿌려 먹는다.

HTML
Hyper Text Markup Language의 약자로 웹 사이트의 구조를 설계할 때 사용하는 언어이다.
CSS
cascading style sheet의 약자로 마크업 요소에 스타일을 정의할 때 사용하는 언어이다.
